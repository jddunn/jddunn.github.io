  • Lucid Dreaming Mask Prototype

    This app uses the NeuroSky MindWave (EEG sensor) with an Ardunio connected to a pulse sensor along with the Processing IDE because of its compatibility with the libraries. The MindWave was taken apart and reassembled with a durable sleeping mask for comfort. Its main function is to induce lucid dreams through playing musical triggers and binaural beats when the program detects the user to be in REM sleep, but it also functions excellently as a real-time EEG / biometric monitor, as the program records data and organizes it into spreadsheets and images for researching purposes.



    Users will eventually be able to send data into the machine during their dreams by controlling eye blinking, and the software will be expanded with binaural beats, dynamic soundscapes, and visual cues (which would affect the environments of the dreams), and a refined, 3D printed mask will be created to house the components. Virtual reality environments (Through using Google Cardboard lenses and a smartphone) will immerse the user in order to stimulate a specific dream. I plan on creating an app for the device for Android / iOS, using Bluetooth to communicate between the two, and putting the final, refined product up on KickStarter.

    See the website in which I publish my research and findings on lucid dreams here.

  • Jasper 2.0 - Talking Virtual Assistant / ChatBot

    This project is expanded off the Jasper speech recognition / synthesis library written in Python. Jasper 2.0 is a portable, wireless system coded running on a Raspberry Pi that has a variety of information processing features, a ChatBot functionality, and the ability to be run locally offline or to be expanded by connecting to online databases.



    This version is housed in a lo-tech robot, where all the components are reattachable and reconfigurable, but the AI software only needs to be run on a Raspberry Pi.

    Currently, Jasper 2.0 can do things like timekeeping, calendar / event management, note taking / transcription, web search, Wikipedia search, IMDB search, picture taking, telling jokes, controlling media, playing games like asking trivia questions, and he has access to various local / cloud databases.

    The most compelling function Jasper 2.0 has is that he will recognize automatically when you are conversing to him, not commanding him, and using the ChatBot (CleverBot) library, he will be able to have a stimulating, intellectual, and often entertaining conversation with you.

    Note: This prototype is a proof-of-concept only; there are no plans to create a more refined robotic enclosure; however, the software of Jasper 2.0 will be improved upon and live on in upcoming projects.


  • Real-Life Pip-Boy

    For this project, I wanted to create a device inspired by the fictional wrist-worn computers called Pip-Boys from the Fallout video game series. There have been many projects dedicated towards recreating perfect Pip-Boys replicas already.

    What I did differently was I chose to take the idea and context of the Pip-Boy and redesign it using components and software that I thought would be practical for an adventurer in an actual post-apocalyptic wasteland. As a result, my redesigned Pip-Boy, although an unrefined prototype, is much more advanced and powerful than the one used in the game story.


    As a practical product for this world, I wanted to have a fully-fledged portable computer that would be modular and re-configurable, but extremely durable and long-lasting. This PipBoy is also meant to be an entirely self-sustaining system in a world where electricity and electronics may have broken down.

    I chose to use a Raspberry Pi B+ and an Arduino Uno in conjunction, with a 5″ touchscreen. The touchscreen sat on top of the pins on the Pi, so the variety of sensors needed to be soldered to the Arduino, all of which are hidden by a detachable shell, so that the Arduino can be used to interface with other components and electronics freely. There is a temperature / humidity sensor enclosed in a protected case along with a compass, a barometer, an SDR antenna, GPS, 300 mbps WiFi, Bluetooth, a triple-axis accelerometer and gyrometer, a motion sensor, a detachable keyboard with an IR receiver and air mouse feature, and an air contaminants / pollutants sensor. It is finally all connected to a large, solar-powered battery pack.

  • BinFace: Trash Bin Robot

    For a project about making a trash bin more appealing, we hooked up LED matrices, a motion sensor, a flex sensor, and a LCD screen to an Arduino. The eyes and the mouth of the face change when someone comes near it and before and after it’s used; it displays banter in a youth-friendly tone.
