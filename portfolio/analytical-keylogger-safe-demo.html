  • freewriter

    Link here

    Freewriter is a web text editor designed as a response to the notorious prevalence of surveilance in urban cities. No data is kept on the site or sent anywhere else. Freewriter has some basic real-time cryptography features, including a one-button text scrambler and a system that automatically encrpyts text randomly based on an oscillating animated UI, leaving the unecrypted text visible for very short periods of time.

    The cryptography is not meant to be that secure, but convenient. Consequently, users can encode and decrypt their texts with the same cipher key anytime they revisit the site. You can, however, make the cipher key as complex as you want with as many different characters as you want.
