  • freewriter

    Link here

    Freewriter is a web text editor designed as a response to the notorious prevalence of surveilance in urban cities. No data is kept on the site or sent anywhere else. Freewriter has some basic real-time cryptography features, including a one-button text scrambler and a system that automatically encrpyts text randomly based on an oscillating animated UI, leaving the unecrypted text visible for very short periods of time.

    The cryptography is not meant to be that secure, but convenient. Consequently, users can encode and decrypt their texts with the same cipher key anytime they revisit the site. You can, however, make the cipher key as complex as you want with as many different characters as you want.

    See the source code on GitHub.


  • New Word Cloud

    Link here

    This is a web app that contains a suite of data analyzing features, including syllable / word / sentence counters, a reading level counter, basic statistical tools, and a visual way of exploring word frequencies. The app also allows users to input up to three different key words, and it will show them all the context sentences surrounding the key word that are found organized so that the exact relationships of the context words to all the others are easily revealed.

    The visualization works only for texts longer than 100 words. Drag and drop any text file, or enter something in the text box and submit. Move your mouse on the x-axis to explore the words on the visualization, and use the ‘UP’ and ‘DOWN’ arrow keys to change the text size. Enter in 1-3 words and submit.

    Obviously, this tool works well to analyze articles and novels and other conventional forms of text, but the program can also parse lists of email addresses based on email domain (gmail, yahoo, msn, etc.) or the email address names. Thus, the site works well with going through data log files for emails as well as phone numbers.

    See the source code on GitHub.


  • Analytical Keylogger (Safe, Virtual Demo)

    Link here

    This site is meant to be a demonstration of the dangers a keylogger with even basic data analyzing features can pose. All text typed on the page is recorded (Not just the message box), with the word frequencies tracked. There is also a potential password finder, which works by checking for phrases that have been entered exactly twice.

    You can download the data saved from the page, although none of it is stored or sent anywhere, so it's all safe! For practical demonstrations, the JavaScript file can be copied and pasted in the console window of any browser, and the data will be stored and printed in the console. The code for this is for security demoing purposes only.

    See the source code on GitHub.

  • Solitary - the Soundscape

    Link here

    I designed this site in Wix to use this as a digital platform for an abstractified, visual soundscape I developed in Unity3D (With the visuals done in After Effects) based on the concept of solitary isolation in Rikers Island, after having done much research on the controversies and events surrounding Rikers. The soundscape was meant to be played and projected on a large screen in a dark room with surround speakers, but I also wanted the experience to be accessible easily online.

    The soundscape is meant to fully transport you into the confines of solitary isolation with dynamic, immersive, and random sounds. Solitary is meant to wear you down with its dark undertones and purposed repetition, as a means of introducing the cruel experience of solitary isolation to the general public, in hopes of raising public awareness.


  • Surreal Text Rearranger

    Link here

    This site was both an artistic tribute and a virtual recreation of the poetic "cut-up technique" for text, developed out of the Dadaist art movement. Input text, and phrases are chopped, rearranged, randomized, and the punctuation removed. You can easily produce many different iterations of Dadaist poetry this way while remaining technically true to its roots.

    See the source code on GitHub.
